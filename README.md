# Etheric World

Etheric World is the parent ecosystem for modular trading machines, research layers, architecture notes, simulation work, and backups.

Phase 1 contains only the Survival Layer and Chimp Layer. DivineOS is the first trading machine inside Etheric World.

## Systems

- `DivineOS/` - Phase 1 MQL5 trading machine.
- `DivineBrain/` - reserved for future cognition and review notes. No AI behavior is active in Phase 1.
- `Architecture/` - ecosystem-level architecture references.
- `Simulation/` - future simulation workspace.
- `Backups/` - preserved source snapshots copied before migration.

## Phase 1 Boundary

Phase 1 does not include Oracle, Ghost Trading, Redis, Python services, or autonomous AI behavior.

The default operating mode is `MICRO_SURVIVAL_MODE` because the current account balance is about $12 and any account under $100 must use micro survival rules.
