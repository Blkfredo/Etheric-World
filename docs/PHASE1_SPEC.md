# Phase 1 Spec

Phase 1 is the foundation layer for DivineOS inside Etheric World.

## Included

- Survival Layer.
- Chimp Layer.
- Micro-account safety.
- Deterministic reason codes.
- CSV logging schema.
- Relay execution preservation.

## Not Included

- Oracle.
- Ghost Trading.
- Redis.
- Python services.
- AI behavior.

## Default Mode

Accounts under `$100` use `MICRO_SURVIVAL_MODE`.

The current account balance is about `$12`, so the default state is `MICRO_SURVIVAL_MODE`.
