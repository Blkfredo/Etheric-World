# Etheric World Overview

Etheric World is the parent ecosystem. It is not the old DivineOS repository and should not be treated as a direct overwrite of existing DivineOS files.

DivineOS lives inside Etheric World as the first trading machine. Other folders are placeholders for future ecosystem expansion, but Phase 1 keeps them inert.

## Current Scope

- Parent ecosystem: Etheric World.
- First machine: DivineOS.
- Active phase: Phase 1.
- Active layers: Survival Layer and Chimp Layer.
- Default balance posture: `MICRO_SURVIVAL_MODE`.

## Explicitly Excluded From Phase 1

- Oracle behavior.
- Ghost Trading.
- Redis.
- Python services.
- AI/autonomous cognition.
