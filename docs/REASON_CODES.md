# Reason Codes

Reason codes are short, stable labels used by logs, state transitions, and execution decisions.

## Survival

- `MICRO_BALANCE_ACTIVE`
- `BALANCE_TOO_LOW`
- `MAX_DAILY_LOSS_REACHED`
- `SPREAD_TOO_HIGH`
- `RISK_TOO_HIGH`
- `LOT_TOO_SMALL`

## Chimp

- `NO_SIGNAL`
- `WEAK_SIGNAL`
- `SIGNAL_READY`
- `SESSION_ALLOWED`
- `SESSION_BLOCKED`

## Execution

- `EXECUTION_ALLOWED`
- `EXECUTION_BLOCKED`
- `ORDER_SENT`
- `ORDER_FAILED`
- `RELAY_ALLOWED`
- `RELAY_BLOCKED`
